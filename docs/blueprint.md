# **App Name**: COG App

## Core Features:

- Worker Profile Management: CRUD operations for worker profiles, including personal details, roles, and permissions.
- Approval Workflow: Tiered approval system for new worker registrations and profile updates.
- Room Reservation: Calendar-based system for booking church rooms, with conflict detection and role-based approval. Includes a list of equipment that can be borrowed.
- Smart Mealstub Generation: Automatic and manual generation of mealstubs based on worker type and schedule.
- QR Code Generation & Scanning: Generates QR codes for mealstub validation and attendance tracking, and enables scanning via device camera.
- Attendance Tracking: QR-based attendance system with clock in/out functionality, linked to mealstub validation.
- User Authentication: Secure API endpoints with JWT authentication for all core modules.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5), evoking trust and stability.
- Background color: Very light grayish-blue (#F0F2F7), providing a calm backdrop.
- Accent color: Lavender (#9575CD), for highlights and interactive elements.
- Font pairing: 'Inter' (sans-serif) for body text and 'Alegreya' (serif) for headings
- Consistent use of Material Design icons, customized in indigo tones.
- Mobile-first, responsive layout for optimal viewing on various devices.
- Subtle transitions and animations to guide user interaction.
